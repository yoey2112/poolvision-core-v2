# Pool Vision OBS Plugin

This plugin integrates Pool Vision Core v2 with OBS Studio for real-time overlay streaming.

## Features

- Real-time game overlays
- Customizable templates
- Multiple streaming platform support
- Advanced drag-and-drop editor
- Performance optimized

## Installation

1. Build the plugin using CMake
2. Copy the plugin files to your OBS plugins directory
3. Restart OBS Studio
4. Add "Pool Vision Overlay" source to your scene

## Usage

1. Start Pool Vision Core v2
2. Connect to OBS in the streaming settings
3. Select your overlay template
4. Start streaming!

## Development

This plugin is built as part of the Pool Vision Core v2 project.
Requires OBS Studio development libraries.

## Requirements

- OBS Studio 29.0+
- Pool Vision Core v2
- Windows 10/11 or Linux
